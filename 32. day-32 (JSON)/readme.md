# 32. Day-32 Json

the json is simply the format that is used to transfer the data over the internet.

- it have no comments.
- javascript object model.
- it can be easily accessed by any language which support object and array ex: js and python.
-  types of values in json
1. String
2. Numbers.
3. null
4. boolean
5. object
6. array

syntax: 

{
    "name":"rishi",
    "age":23,
    "client":{
        "id":"a101"
        "address":"#123"
    },
    "score":[
            {"f1":32,
            "f2":20,
            "f3":12
            }
        ]
}

the key must be in string value.
